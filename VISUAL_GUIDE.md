## Performance Tips

### For Best Experience
1. Use a modern browser (Chrome, Firefox, Safari, Edge)
2. Enable hardware acceleration
3. Use a device with decent GPU
4. Desktop recommended for full effects

### If Performance Issues
1. Disable cursor particles (they auto-disable on mobile)
2. Use light theme (slightly better performance)
3. Enable "Reduce Motion" in OS settings (animations will simplify)

## Browser DevTools Tips

### To See Animations in Slow Motion
1. Open DevTools (F12)
2. Press Ctrl+Shift+P (Cmd+Shift+P on Mac)
3. Type "animations"
4. Select "Show Animations"
5. Adjust playback speed

### To Check Performance
1. Open DevTools (F12)
2. Go to Performance tab
3. Record while interacting
4. Check for 60fps (green line)
